Bienvenido a Lopple

Bienvenido a la documentación oficial del bot Lopple. Aquí podrás encontrar información detallada sobre su funcionamiento y comandos disponibles.

Resumen rápido

Lopple es un bot de Discord desarrollado en Python, diseñado para ser el corazón de un torneo de Splatoon 3 llamado Stellaris Phantasm.

  • 🎯 Gestión avanzada de equipos con roles automáticos.
  • 🧩 Sistema de comandos personalizados con respuestas visuales.
  • ⚙️ Integración con APIs para funcionalidades extendidas.
